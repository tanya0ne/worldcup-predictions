# Прогнозы ЧМ-2026

Игра-прогнозы на Чемпионат мира 2026 для двоих. Vite + React + TypeScript + Tailwind, данные в Supabase, хостинг GitHub Pages.

- Прогнозы на счёт каждого матча (3 очка точный счёт / 2 разница / 1 исход).
- Ставка на чемпиона до старта турнира (+25 очков).
- Таблица очков на двоих.

Расписание и результаты матчей пишет робот из приватного репозитория `tanya-automation` (источник: fixturedownload.com) в ту же базу Supabase.
